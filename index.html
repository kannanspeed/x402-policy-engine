# 3 lines to add autonomous payments to your AI agent

from x402_policy import PolicyConfig, x402PolicyClient

policy = PolicyConfig(daily_budget=5.0, per_tx_max=0.10, allowlist=["api.coinbase.com"])
client = x402PolicyClient(wallet_private_key="0x...", policy_config=policy)

await client.pay("api.coinbase.com", 0.05, "ETH price feed")  # Auto-approved under policy

How It Works

1

Agent requests payment

Your AI agent calls client.pay("api.openai.com", 0.05) — just like making an API request.

2

Policy engine evaluates

Checks: within daily budget? under per-tx limit? recipient allowed? time window OK? ERC-8004 reputation sufficient?

3

x402 payment executes

If approved: on-chain USDC payment via x402 relay. If denied: returns reason, no tx mined.

4

You stay in control

Dashboard shows all transactions. Kill switch. Adjust limits. Add recipients. All in real-time.
